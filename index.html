A collection of specialized calculators for data analysis

Select a calculator below to perform specific statistical and mathematical calculations. Each tool is designed to simplify complex computations with an intuitive interface.

CMH Calculator

Cochran-Mantel-Haenszel Statistics

The Cochran-Mantel-Haenszel calculator is used to analyze the association between two binary variables, while controlling for one or more confounding variables. This is particularly useful in stratified analysis.

Key Features:

  • Calculate CMH common odds ratio estimate
  • Compute Mantel-Haenszel statistics
  • Stratified analysis with multiple layers
  • Confidence interval calculation
  • Export results for reporting

Pairwise Distance Calculator

3D Distance Calculations

Calculate Euclidean distances between multiple points in three-dimensional space. This tool is ideal for spatial analysis, molecular modeling, or any application requiring precise distance measurements in 3D.

Key Features:

  • Calculate distances between multiple 3D points
  • Support for batch processing of point pairs
  • Visual representation of points in space
  • Export distance matrix for further analysis
  • Customizable coordinate input formats

Statistical Power Calculator

Sample Size & Power Analysis

Determine the appropriate sample size for your study or calculate statistical power for existing data. This tool helps researchers design more effective studies and interpret results accurately.

Key Features:

  • Calculate sample size for various tests
  • Power analysis for t-tests, ANOVA, and more
  • Effect size estimation
  • Interactive power curves
  • Save and export analysis parameters
